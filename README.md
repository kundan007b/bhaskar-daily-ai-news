## ğŸš€ Quick Start

### Prerequisites

- Jekyll 4.0+
- Ruby 3.0+
- Git

### Local Development

```bash
# Clone repository
git clone https://github.com/kundan007b/bhaskar-daily-ai-news.git
cd bhaskar-daily-ai-news

# Install dependencies
bundle install

# Serve site locally
bundle exec jekyll serve
# Visit http://localhost:4000
```

### Publishing Posts

1. Create markdown file in `_posts/` with format: `YYYY-MM-DD-title.md`
2. Add required front matter:
   ```yaml
   ---
   layout: post
   title: "Your Article Title"
   author: author-slug
   categories: [AI, Startups]
   image: /assets/images/featured.jpg
   lang: en
   date: 2025-12-14
   ---
   ```
3. Push to GitHub â†’ Automatic build & deploy

### GitHub Pages Deployment

- Automatic on every push to `main` branch
- No manual setup needed
- Site builds via GitHub Actions

## ğŸ“° Google News Compliance

### XML Sitemaps

**news-sitemap.xml** - Google News specific
- Posts from last 48 hours only
- Includes publication name, language, keywords
- Auto-regenerated on each build

**sitemap.xml** - Standard sitemap
- All site URLs
- Generated by jekyll-sitemap plugin
- Includes pages and posts

### Structured Data

**NewsArticle JSON-LD** (`_includes/schema-jsonld.html`)
- Automatically generated per post
- Fields: headline, datePublished, dateModified, author, publisher, image
- Proper timezone handling (Asia/Kolkata)

### E-E-A-T (Expertise, Authoritativeness, Trustworthiness)

- **Author Pages** - Individual bios with credentials
- **About Page** - Clear publication info
- **Editorial Policy** - Transparent editorial standards
- **Contact Info** - Direct communication channel

### SEO Configuration

**robots.txt** - Directs crawlers
- Allow all content
- Reference to sitemaps

**_config.yml** - Jekyll plugins:
- jekyll-seo-tag
- jekyll-feed
- jekyll-sitemap
- jekyll-paginate
