# Social Media Graphics Templates - Bhaskar Daily News

Create consistent, professional social media graphics for every post.

## Brand Colors

```
Primary Purple: #667eea
Secondary Purple: #764ba2
Accent Color: #f093fb
```

## Typography

- **Headings**: Poppins Bold or Inter Bold
- **Body**: Inter Regular or Roboto
- **Accent**: Poppins SemiBold

## Image Sizes

### Twitter/X
- **Post Image**: 1200x675px (16:9)
- **Header**: 1500x500px

### LinkedIn
- **Post Image**: 1200x627px
- **Header**: 1584x396px

### Facebook
- **Post Image**: 1200x630px
- **Cover**: 820x312px

### Instagram
- **Feed Post**: 1080x1080px (square)
- **Story**: 1080x1920px (9:16)
- **Carousel**: 1080x1080px per slide

- **Colors**: Orange, purple gradients

## Automation Workflow

### Create 5 Templates in Advance
1. News announcement
2. Data/stat
3. Quote
4. List/tips
5. Question/engagement

### Weekly Batch Creation (1 hour)
1. Open all 5 posts for the week
2. For each post, choose template
3. Customize with post details
4. Download all in one go
5. Schedule with Buffer/Hootsuite

### Tools for Scheduling
- **Buffer**: Free for 3 accounts
- **Hootsuite**: Free tier available
- **Later**: Best for Instagram
- **TweetDeck**: Free for Twitter

## Instagram-Specific Tips

### Carousel Posts
Create 5-10 slides per post:
1. **Slide 1**: Hook (big statement)
2. **Slides 2-8**: Key points (one per slide)
3. **Last Slide**: CTA + website link

### Story Templates
- Question stickers for engagement
- Poll stickers for opinions
- Link stickers to blog
- Countdown for events

### Reels Ideas
- News recap (60 sec)
- Quick tips (15 sec each)
- Behind the scenes
- Trending audio with news overlay

## Twitter/X Specific Tips

### Thread Graphics

- **Image 1**: Main headline

- **Image 5**: CTA

### Quote Tweets

- Expert opinions

## Performance Tracking

### What to Track
- **Engagement rate** per template type
- **Click-through rate** to website
- **Best posting times** per platform
- **Top performing topics**

### Monthly Review
1. Which templates got most engagement?
2. Which colors worked best?
3. What content types drove traffic?
4. Adjust templates based on data

## Brand Consistency Checklist

Every graphic should have:
- [ ] Brand colors used
- [ ] Readable fonts (mobile-friendly)
- [ ] Website URL
- [ ] Category indicator
- [ ] High contrast for readability
- [ ] Proper image size for platform
- [ ] Compressed for fast loading

## Quick Start Guide

### First Week
1. Create 5 base templates in Canva
2. Make 3 variations of each (15 total)
3. Create one graphic per post
4. Test on all platforms

### First Month
1. Track which templates perform best
2. Refine top 3 templates
3. Batch create graphics weekly
4. Build graphics library

### Ongoing
1. Update templates seasonally
2. Test new styles monthly
3. A/B test different designs
4. Keep winning templates
